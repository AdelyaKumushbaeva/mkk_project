# Информационная система ООО МКК "Денежная единица"

Данный проект представляет собой программный модуль (WPF App) на языке C# для автоматизации учета и расчета микрозаймов.

### Основные возможности ИС:
- **Регистрация договоров:** Ведение единого реестра клиентов и выданных займов.
- **Финансовый калькулятор:** Автоматический расчет суммы возврата по процентной ставке 0.8% в день.
- **Валидация данных:** Контроль корректности заполнения полей при сохранении информации в базу данных.

### Технический стек:
- **Язык программирования:** C#
- **Графический интерфейс:** WPF (.NET)
- **База данных:** Microsoft SQL Server
- **Технологии доступа к данным:** Entity Framework
- **Контроль версий:** Git / GitHub
